Como plotar Gráfico de Múltiplas Barras com Pandas e Matplotlib¶

(How to plot Multiple Bars Chart with Pandas and Matplotlib)¶

Pandas e Matplotlib são uma das bibliotecas mais famosas dentro do escopo de ferramentas de Data Science. Essas duas bibliotecas, juntamente com o Numpy e o Sklearn, já formam um PyData Stack suficientemente potente para a realização de diversos projetos.

O Matplotlib é talvez a principal biblioteca destinada a criação e visualização de gráficos, dada a sua vasta gama de funções, facilidade de uso, tipos de gráficos, recursos de customização, entre outras vantagens. Contudo, nem tudo são fantasias. Apesar de ser uma ótima ferramenta, como qualquer outra, também tem suas limitações.

Eu ainda estou engatinhando em Data Science, mas esta é uma área que eu tenho bastante interesse e estou me dedicando muito a aprender. Contudo, durante meus estudos, me deparei com um problema, que aparentemente é simples, mas que não encontrei nenhuma solução que realmente o resolvesse do jeito que eu desejava.

O tal problema era: Plotar um Gráfico de Múltiplas Barras com o Matplotlib.

Então, resolvi procurar algumas soluções. Olhei na documentação do Matplotlib e de fato encontrei uma resposta para isso, porém, que não me agradou muito. O formato de construção deste tipo de gráfico era uma quantidade enorme de código, além de pouco funcional, isto é, você precisa ficar adaptando todo o código aos seus dados em vez de ser algo simples. Dá uma olhada:

https://matplotlib.org/examples/api/barchart_demo.html

Bom, continuei procurando outras soluções no StackOverflow, fóruns, blogs, mas não encontrei nenhuma função ou código que ajudasse a simplesmente plotar um gráfico de múltiplas barras a partir de um dataset qualquer.

Após um bom tempo de pesquisa descobri que o próprio Pandas tem uma função construída com base no Matplotlib que plota um gráfico de barras múltiplas bem estruturado, porém para isso, os dados precisam estar organizados em um Dataframe.

Então, resolvi criar uma pequena função que ajude a simplesmente selecionar o eixo X e Y do dataset a serem plotados, já organizando o dataframe através da soma e correspondência entre os valores, e deixar todo resto do trabalho para o Pandas.

Enfim, chega de enrolação. É essa função que vou deixar abaixo e explicar um pouco como ela funciona antes de começarmos a testá-la. Bora lá!

Importando as Bibliotecas¶

Como de costume, vamos iniciar importando as bibliotecas que serão usadas. Neste caso serão:

  • Pandas, necessário para plotar o gráfico.
  • Matplotlib, customização dos gráficos.
  • E o módulo Random, que usaremos apenas para criar um dataset para teste.

Ou seja, para utilizar a função só será necessário mesmo o Pandas.

import pandas as pd
import random
import matplotlib.pyplot as plt
%matplotlib inline
# Esse comando '%matplotlib inline' é específico do Jupyter Notebook, ele serve para que o plot dos gráficos do Matplotlib aconteça dentro do próprio Jupyter, sem a necessidade de abrir uma nova janela para isso.

Função Multbars - Como plotar um Gráfico de Múltiplas Barras¶

Bom, essa função tem apenas o objetivo de auxiliar no plotagem dos gráficos. Ela será responsável por filtrar e somar os valores dos dados, com base nas listas que passamos para o eixo x e y como parâmetros da função.

Então, no final essa função apenas gera um novo dataframe, só que agora pronto para ser usado com o Pandas e plotado nos gráficos de múltiplas barras.

Para não poluir muito o código, vou adicionar comentários numerando pontos da função, e deixarei abaixo a explicação de cada ponto.

Mas, não precisa ler toda essa explicação. Se quiser, pode pular para frente e já ir logo para a parte aplicação! ;)

Entendendo a Função Multbars():¶

  • 1) Criação do nosso Dataframe Inicial e Resultante
  • 2) Aqui, passamos o método .value_counts() na lista do eixo y. Este método irá contar a quantidade de registros e gerar uma nova lista com a quantidade total para cada registro único. Então, nós adicionamos o método .index, que selecionará apenas os indexs da lista, e executamos um loop for que percorrerá todos valores. Então, iremos adicionar os indexs ao dataframe resultante como keys(chaves) e atribuir uma lista vazia como value(valor) para cada key.

  • 3) Agora, nós faremos a mesma coisa que fizemos com o eixo y, só que no eixo x. Vamos passar o método .value_counts() no eixo x, e passar o loop for pela lista gerada por ele, pegando apenas os indexs com o método .index. Após isso, vem a parte mais importante.

    A variável values, vai armazenar o filtro do dataframe inicial(initial_dataframe). Este filtro funciona assim: o index que vem do loop for, filtra o dataframe inicial por ele. Ou seja, serão selecionadas apenas as linhas que contêm aquele index como valor. Então, pegamos a coluna do eixo y e contabilizamos os seus valores.

  • 4) Vamos percorrer cada chave do dataframe resultante e checar com uma condicional if, se alguma chave não existe nos indexs da lista values. E por que fazemos este comando? Fazemos para checar se alguma chave do dataframe resultante não existe nesta lista de indexs, pois caso não exista, adicionaremos a variável values a chave que não existe, para que ela passe a existir e atribuiremos o valor 0, justamente por este ser o valor da quantidade do registro que não existia.

    Assim, fazemos com que ao final de todo o processo, a quantidade de valores da variável que esta sendo plotada no gráfico sejam iguais. Algo que é requisito obrigatório das funções de plotagem.

  • 5) No quinto passo, vamos percorrer os indexs da variável values e adicionar ao dataframe resultante, os valores indicados para cada chave.
  • 6) No sexto e penúltipo passo, vamos apenas adicionar ao dataframe resultante os valores do eixo x, então criamos uma key com o nome do eixo x, e atribuímos a chave os valores, que no caso são os indexs da própria variável.
  • 7) Pronto! O último passo é dar um return no dataframe resultante passando também a função pd.DataFrame() para deixar ele nos moldes complestos do Pandas. E foi! A função está pronta!
In [8]:
def multbars(x, y):
    # 1)
    initial_dataframe = pd.DataFrame({x.name: x, y.name: y})
    result_dataframe = {}
    
    # 2)
    for index in y.value_counts().index:
        result_dataframe[index] = []

    # 3)
    for index in x.value_counts().index:
        values = initial_dataframe.loc[initial_dataframe[x.name] == index][y.name].value_counts()
        
        # 4)
        for key in result_dataframe:
            if key not in values.index:
                values[key] = 0

        # 5)
        for index in values.index:
            result_dataframe[index].append(values[index])
    # 6)
    result_dataframe[x.name] = x.value_counts().index
    
    # 7)
    return pd.DataFrame(result_dataframe)

Como sabemos, quem plotará o gráfico de múltiplas barras será o Pandas, essa função serve apenas para facilitar a nossa vida, pois conseguimos apenas selecionar as colunas do dataset e já plotar automaticamente. Então, todo o trabalho da função é somar os valores dos registros e organizar no dataset que será plotado.

Legal! Já temos o nosso Dataset pronto e já criamos e testamos a função multbars(). Agora, vamos ver alguns exemplos de sua aplicação com o Pandas.

Bom, o gráfico será plotado através do método .plot.bar() do Pandas. Este método precisa ser aplicado em um Dataframe, que será o resultado da função multbars(). Veja o exemplo abaixo!

(Estamos utilizando a função plt.show() do Matplotlib para deixar a visualização do gráfico mais limpa.)

In [11]:
multbars(data['Faixa Etária'], data['Sexo']).plot.bar()
plt.show()

Legal! O gráfico foi plotado com múltiplas barras! Show! Porém, ele está estranho, o eixo x está levando uma numeração e não os valores do eixo x que selecionamos para plotar.

Isso é porque para plotar o gráfico com o eixo x organizado, precisamos passar um parâmetro no método .plot.bar(). Este parâmetro vai indicar quais os valores do dataframe entrarão no eixo x. Olhe o exemplo a seguir.

In [12]:
multbars(data['Faixa Etária'], data['Sexo']).plot.bar(x='Faixa Etária')
plt.show()

Beleza, agora os valores do eixo x estão corretos. Ainda sim tem um pequeno problema, está tudo escrito na vertical! Para resolver isso, vamos passar mais um parâmetro no método, o rot=0. Vamos ver como ficou!

In [13]:
multbars(data['Faixa Etária'], data['Sexo']).plot.bar(x='Faixa Etária', rot=0)
plt.show()

Legal, conseguimos rotacionar os valores do eixo x, mas os textos estão todos embaralhados por causa da largura do gráfico. Bom, como você já deve suspeitar, para ir fazendo os ajustes que precisamos basta passar os parâmetros comuns do Matplotlib, até porque, apesar de ser um método do Pandas, foi construído em cima do Matplotlib.

Então, vamos passar o parâmetro figsize=() e ajustar o tamanho do gráfico.

In [14]:
multbars(data['Faixa Etária'], data['Sexo']).plot.bar(x='Faixa Etária', rot=0, figsize=(14, 5))
plt.show()

Pronto, agora sim! Nosso gráfico atingiu o ponto com o mínimo necessário para uma visualização decente.
