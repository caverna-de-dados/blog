Python, Newton e Fractais

Diego Zurita | Sun 12 April 2020

Introdução¶

Continuando a série sobre métodos númericos, hoje vou escrever um pouco sobre o método de newton (também conhecido como método de Newton–Raphson). A idéia central deste método é bem simples porém elegante, e pode ser aplicada em diferentes problemas, como por exemplo em otimização contínua, no "passo de newton".

Problemática e solução de newton¶

O problema que estamos interessados ainda é encontrar 0 de uma função $f(x)$. Para tal, vamos gerar uma sequência de $x_k$, a partide de um chute inicial $x_0$, tal que a cada iteração, o valor de $f(x_k)$ está mais próximo da raiz, ou seja, $f(x_k)=0$ quando $k \to \infty$. (Para um $x_0$ próximo o suficiente da raiz).

O termo $x_k$ é dado pela forma:

$$ x_{k} = x_{k-1} - \frac{f(x_{k-1})}{f'(x_{k-1})} $$

Ou de forma equivalente,

$$ x_{k+1} = x_k - \frac{f(x_k)}{f'(x_k)} $$

Implementação em python¶

A implementação consiste em uma classe $MetodoNewton$ com alguns métodos úteis, são eles:

  • __init__: recebe o chute inicial $x_0$, um número $maximo\_de\_iteracoes$, um $eps$ que diz quão longe da raiz podemos parar, e as funções $f$ e $Df$ , $f(x)$ e sua derivada, respectivamente.

  • proximoxk: retorna o $x{k+1}$ da equação acima dado o paramêntro $x=x_k$

  • itera: aplica o método de newton sucessivamente até que o $eps$ seja respeitado

  • obetem_reta_tangente: obtem a reta tangente próxima de $x_k$ nos pontos $x$

  • gera_animacao: gera uma animação para mostrar como o $x_k$ esta convergindo para a raiz

  • plot_x: mostra um gráfico de $f(x)$

  • convergiu: verficar se o método convergiu ou não, depois de chamar o método $itera$

## Bibliotecas de exibição
import matplotlib.pyplot as plt
import numpy as np
import seaborn as sns
from matplotlib import  animation
from IPython.core.display import display, HTML

sns.set()

class MetodoNewton():
    def __init__(self, x_0, maximo_de_iteracoes, eps, f, Df):
        self.x_0 = x_0
        self.maximo_de_iteracoes = maximo_de_iteracoes
        self.eps = eps
        self.sequencia = [self.x_0]
        self.raiz = None
        self.f = f
        self.Df = Df
        self.num_itr = 1

    def proximo_xk(self, x):
        return x - self.f(x)/self.Df(x)
        
    def itera(self):
        x_k = self.x_0
        self.sequencia = [self.x_0]
        self.num_itr = 1
        
        while( abs(self.f(x_k)) > self.eps ):
            x_k = self.proximo_xk(x_k)
            self.sequencia.append(x_k)
            self.num_itr += 1
            
        #Convergiu, portanto encontrou uma raiz.
        if abs(self.f(x_k)) <= self.eps: 
            self.raiz = x_k
        
    def obtem_reta_tangente(self, x, x_k):
        return self.Df(x_k) * (x - x_k) + self.f(x_k)
            
    def gera_animacao(self, x, delta=3):
        fig = plt.figure()
        ax1 = fig.add_subplot(1,1,1)
        
        def animate(i): 
            ax1.clear()
            x_k = self.sequencia[i]
            x_para_tangente = np.linspace(x_k-delta, x_k+delta, 10)
            
            ax1.plot(x, self.f(x))
            ax1.plot(x_para_tangente, self.obtem_reta_tangente(x_para_tangente, x_k))
            ax1.hlines(y=0, xmin=x[0], xmax=x[len(x)-1], linestyles="dashed")
            plt.scatter(self.proximo_xk(x_k), 0, c="r")
            
        ani = animation.FuncAnimation(fig, animate, frames=len(self.sequencia), interval=500)
        video_tag = ani.to_html5_video()
        plt.close()
        display(HTML(video_tag))
        
    def plot_f(self, x):
        plt.figure(dpi=400)
        plt.title("Forma de f(x)")
        plt.plot(x, self.f(x))
        plt.hlines(y=0, xmin=x[0], xmax=x[len(x)-1], linestyles="dashed")
        plt.ylabel("f(x)")
        plt.xlabel("x")
        plt.show()
    
    def convergiu(self):
        return self.raiz is not None
        
    def __str__(self):
        if self.convergiu():
            return "A raiz encontrada foi {} e foi encontada depois de {} iterações".format(self.raiz, self.num_itr)
        else:
            return "Não encontrei raiz alguma :/"

def f_exemplo(x):
    return 3 * np.power(x, 3) - 2

def Df_exemplo(x):
    return 9 * np.power(x, 2)

newton_exemplo = MetodoNewton(13, 100, 0.001, f_exemplo, Df_exemplo)

x = np.linspace(-5, 5, 100)
newton_exemplo.plot_f(x)

Visualmente é possivel ver que há uma raiz próximo ao $x=0$, mas vamos usar o método de newton para encontrar essa raiz:

newton_exemplo.itera()
print(newton_exemplo)
x = np.linspace(-9, 18, 100)

A raiz encontrada foi 0.8736897604312023 e foi encontada depois de 10 iterações

Na animação vemos como o método se comporta. A bolinha vermelha é o $x_{k+1}$, ou seja, é o próximo iterando, é póssivel ver que a partir de certo instante, ela mal se mexe. (quando a reta tangente esta "deitada").

Fractais¶

Mas a pergunta que surge aqui é: como com esse método podemos chegar em um fractal? O pulo do gato aqui são as bacias de convergência. Bacias de convergência são valores possiveis de $x$ cujo o metódo converge para a mesma raiz. Dificil? Vamos ao exemplo anterior!

Por uma inspeção rápida sabemos que $f(x)$ tem 3 raizes, mas na implementação só encontramos 1, onde estão as outras duas? Elas moram no plano complexo, e dependendo do $x_0$ escolhido, encontramos uma raiz diferente. Se visualizamos de uma maneira esperta, vamos encontrar fractais!!! (Uhuuuuul \o/)

Essa maneira esperta se assemelha bastante a visualização das "fronteiras de decisão" de classificação.

Para fazer isso, vamos imaginar o plano complexo, como na imagem abaixo por exemplo:

Então se para cada ponto desse plano aplicarmos o método de newton, ele vai convergir pra uma raiz do polinômia, e para cada raiz distinta atribuimos uma cor.

Exemplificando, se partirmos do ponto $x_0 = 8 + 0i$, vamos encontrar a raiz $r=0$, enão atribuimos uma cor para essa raiz.

Depois de fazer isso para todos os pontos do plano, obetendo a matriz de cores gerada e exibindo como uma imagem, vamos ter um fractal. (Woooooooooow, massa!)

Agora, vamos implementar isso, primeiro vou precisar alterar um pouco a clase original $MetodoNewton$, para poder reaproveitar melhor e também vou criar uma classe chamada $Fractal$, que vai ficar resposável por gerar a imagem:

class MetodoNewton2():
    def __init__(self, maximo_de_iteracoes, eps, f, Df):
        self.raiz = None
        self.maximo_de_iteracoes = maximo_de_iteracoes
        self.eps = eps
        self.f = f
        self.Df = Df
        
    def itera(self, x_0):
        x_k = x_0
        
        while( abs(self.f(x_k)) > self.eps ):
            x_k = x_k - self.f(x_k)/self.Df(x_k)
            
        if abs(self.f(x_k)) <= self.eps: #Convergiu, portanto encontrou uma raiz.
            self.raiz = x_k
            
        return x_k if self.convergiu() else None
    
    def convergiu(self):
        return self.raiz is not None
    
class Fractal():
    def __init__(self, pixel_superior, pixel_inferior, qtd_pixels, newton):
        self.pixel_superior = pixel_superior
        self.pixel_inferior = pixel_inferior
        self.qtd_pixels = qtd_pixels
        self.matriz = np.empty(qtd_pixels)
        self.raizes = []
        self.cor_nao_convergente = 1
        self.newton = newton
        
    def obtem_cor_da_raiz(self, raiz):
        if raiz is None: return self.self.cor_nao_convergente
        
        indice_raiz = -1
        
        for i in np.arange(len(self.raizes)):
            if np.abs(raiz - self.raizes[i]) < 0.01:
                indice_raiz = i
                break
                
        if indice_raiz == -1:
            self.raizes.append(raiz)
            return len(self.raizes) + 1
        else: return indice_raiz + 1
                
        
    def gera_imagem(self, dpi=100):
        x_pixels = np.linspace(self.pixel_superior[0], self.pixel_inferior[0], self.qtd_pixels[0])
        y_pixels = np.linspace(self.pixel_superior[1], self.pixel_inferior[1], self.qtd_pixels[1])
        
        for i in np.arange(len(x_pixels)):
            for j in np.arange(len(y_pixels)):
                raiz_encontrada = self.newton.itera(complex(x_pixels[i], y_pixels[j]))
                cor = self.obtem_cor_da_raiz(raiz_encontrada)
                self.matriz[i][j] = cor
                
        plt.figure(figsize=(self.qtd_pixels[0]/dpi, self.qtd_pixels[1]/dpi), dpi=dpi)
        plt.imshow(self.matriz)
        plt.xticks([])
        plt.yticks([])
        plt.show()

A alteração da classe $MetodoNewton$ foi no método itera, que recebe um $x_0$, isso vai facilitar usar a mesma classe para diferentes pontos.

Ja a classe $Fractal$ possui 4 métodos:

  • __init__: recebe a cordenada do pixel superior direito e pixel inferior direito, com esses dois pixel podemos saber o tamanho da imagem. Também recebemos quantos pixels queremos na horizontal e quantos queremos na vertical e por fim qual $MetodoNewton2$ vamos usar para gerar essa imagem

  • obtem_cor_da_raiz: dada uma raiz, retorna a cor assciada a ela, vale ressaltar que o método não convergir, devemos retornar uma cor padrão

  • gera_imagem: fica resposável por gerar uma imagem com os parametros acima.

Chegou a hora de testar, vamos ver como ficaria o fractal associado a função do exemplo anterior:

Conclusão¶

Primeiro vimos como implementar o método de newton para encontrar 0 de funções e que sua convergência é rapido. Vale ressaltar que para que o método convirga, o chute inicial deve estar próximo o suficiente da raiz. Em seguinda vimos como conectar essas idéia para gerar imagens e ver que essas são fractais.
