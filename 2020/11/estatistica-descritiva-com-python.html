Estatística Descritiva com Python (Parte 1)

Hugo Trigueiro | sáb 21 novembro 2020

A Estatística pode ser definida de forma simplificada, como: uma subárea da Matemática que busca organizar, descrever, resumir e interpretar a informação, através da coleta, descrição e análise de dados.

Assim, ela se torna uma poderosa ferramenta que nos auxilia durante o processo de exploração de dados.

Neste post, vamos ver através da linguagem Python alguns dos conceitos mais básicos da Estatística Descritiva (uma das subdivisões da Estatística, responsável principalmente pelo resumo e descrição da informação).

A intenção é ter uma visão básica e simples da aplicação dos conceitos e não necessariamente aprofundá-los em seus detalhes.

Para isso, vamos utilizar alguns dos principais pacotes para Python voltados à análise de dados. São eles: Pandas e Matplotlib.

Para aplicar os conceitos, foi criado um Dataset fictício. A base de dados se refere a um grupo de candidatos a uma vaga de Cientista de Dados em uma empresa. O Dataset está disponível no respositório deste post no Github do Caverna de Dados.

Todas as demais referências e links utilizados na construção deste post estarão ao final do texto.

Antes de colocarmos a mão na massa, precisamos importar os pacotes que utilizaremos. Como já sabemos, vamos usar os pacotes:

  • Matplotlib. O Matplotlib é uma das principais bibliotecas para plotar gráficos em Python, com ele é possível plotar gráficos de linha, barras, dispersão, setor, histogramas e até graficos em 3D.

Distribuição de Frequências¶

Quando falamos de Estatística Descritiva, estamos buscando compreender melhor o comportamento das variáveis com que estamos trabalhando. Um dos conceitos mais básicos empregados nesta tarefa é a noção de Distribuição de Frequências, que procura resumir o comportamento de uma variável pela contabilização da frequência de observações contidas nela.

Por exemplo, podemos querer saber a quantidade de candidatos formados em Estatística e Economia. Para isso, precisamos da distribuição de frequências da variável 'Formacao'.

Para obtermos a distribuição de frequências de uma variável com o Pandas, podemos simplesmente utilizar o método .value_counts(), que irá retornar a frequência que cada observação única possui naquela variável.

In [11]:
print('Quantidade de Candidatos por Curso de Formação: ')
df['Formacao'].value_counts()

Se quisermos saber qual a distribuição de frequências relativa, isto é, o quanto a quantidade de cada observação única representa no total de observações da variável, podemos passar o parâmetro .value_counts(normalize=True)

Uma outra forma de olhar para a distribuição de frequências de uma variável, é através da distribuição de frequências acumulada. A distribuição de frequências acumulada indica quantos elementos ou que percentual deles estão abaixo de determinado valor.

Aplicando o método .cumsum() juntamente com o .value_counts(), conseguimos facilmente obter uma série cumulativa das frequências.

Por exemplo, vamos analisar a distribuição de frequências acumulada dos cursos de formação dos candidatos, comparando com a sua distribuição de frequências absoluta:

# Distribuição de Frequências Acumulada
dist_freq_abs_acu = df['Formacao'].value_counts().cumsum()

pd.DataFrame({'Distribuição de Frequências Absoluta - Formacao': dist_freq_abs,

Gráficos¶

Gráficos são representações ilustrativas extremamente importantes na Estatística Descritiva. Eles são responsáveis pela visualização dos dados de forma ilustrativa, auxiliando assim na análise e interpretação dos mesmos.

Há muitos tipos de gráficos, contudo temos alguns que são mais utilizados.

  • Gráficos para Variáveis Qualitativas: Gráficos de Barras e de Setores ("Pizza")

Quando estamos analisando uma variável qualitativa, os Gráficos de Barras e de Composição de Setores são alguns dos mais conhecidos. Eles são ótimos para a visualização da distribuição de frequências de uma variável qualitativa.

Por exemplo, se quisermos visualizar a distribuição de frequências da variável 'Principal Linguagem' em um gráfico de barras, podemos utilizar o pacote Matplotlib pra fazer isso.

O Matplotlib possui uma imensa variedade de gráficos que podem ser usados. No caso do gráfico de barras, podemos plotá-lo através do objeto plt.bar(), onde precisamos passar pelo menos dois parâmetros com os eixos X e Y consecutivamente.

Vamos criar duas variáveis, uma chamada linguagem e outra frequencia. A variável linguagem irá conter os indexes do dataframe de distribuição de frequências, o index do dataframe, neste caso, são os próprios valores da variável. Enquanto que as frequências de cada valor serão armazenados na variável frequencia.

Assim, podemos adicionar os métodos .index e .values à distribuição de frequência da variável 'Principal Linguagem' obtida com df['Principal Linguagem'].value_counts() para recebermos dois arrays com os valores e a quantidade de vezes que eles aparecem na variável.

In [15]:
linguagem = df['Principal Linguagem'].value_counts().index
frequencia = df['Principal Linguagem'].value_counts().values

Basta passarmos uma string dentro destas funções para plotarmos um novo gráfico com estas informações.

Utilizando a função plt.pie() e passando pelo menos três parâmetros: x=frequencia, labels=sexo e autopct='%2.2f%%' com a string '%2.2f%%'(essa string se refere ao formato em que o valor com o percentual de cada setor tem na variável é apresentado, neste caso, estaremos usando um formato com valores flutuantes do tipo: 'aa.aa%'), podemos facilmente gerar esta visão.

Não podemos esquecer de colocar o título e nomes dos eixos X e Y, além da função plt.show() para o modo display.

Medidas Resumo¶

Uma etapa fundamental na interpretação e conhecimento dos dados que estamos trabalhando é observar métricas/medidas resumo das variáveis, ou seja, medidas que permitam reduzir os dados a alguns valores representativos de toda a variável.

Medidas de Posição/Localização¶

As medidas de posição ou medidas de localização visam fornecer informações sobre, de forma geral, onde a maior parte dos dados está se concentrando. Uma forma de analisar um variável por medidas de posição, é utilizando as medidas de posição central que forcem uma informação resumida da tendência central dos dados. Temos algumas medidas famosas com este propósito, como a Média Aritmética, Mediana e Moda.

  • Média Aritmética

    $$\LARGE\overline{M} = \frac{\sum_\limits{i=1}^{n} X_i}{n}$$

    A Média Aritmética como descrita na fórmula acima, nada mais é do que a soma de todos os valores de uma variável divido pela quantidade de valores somados. Por exemplo, se temos a seguinte lista [2, 4, 6, 8], para sabermos a média deste conjunto de valores, precisamos somá-los $(2 + 4 + 6 + 8 = 20)$ e depois dividí-los pela quantidade de valores $(\frac{20}{4} = 5)$.

    Para obtermos a média aritmética de uma série em Python usando o Pandas, basta usarmos o médodo .mean(). Vamos obter a média das idades da tripulação do Titanic.

In [25]:
# Média Aritmética dos salários preteridos pelos candidatos
df['Pretensao Salarial (mil)'].mean()
Out[25]:
6.205

  • Média Ponderada $$\LARGE\overline{M_p} = \frac{\sum\limits_{i=1}^{n} X_ip_i}{\sum\limits_{i=1}^{n} p_i}$$

    A Média Ponderada é a soma de cada valor de uma série, multiplicado por seu "peso", e dividido pela soma dos pesos. O que chamamos de "pesos" em uma Média Ponderada, pode ser explicado basicamente como valores que atribuem diferentes "relevâncias" para cada valor que irá compor a média calculada.

    Por exemplo, suponhamos que a empresa estabeleça que os candidatos com média final maior ou igual a 7 passarão para a próxima fase do processo seletivo. Se um candidato obteve notas 8, 8, 4 e 7 nas provas de Matemática/Estatística, Programação, Português e Negócios respectivamente, então a sua média final (aritmética) é igual a 6,75 ($\frac{8+8+4+7}{4} = 6,75$) e então ele não passaria de fase. Contudo, sabendo que as habilidades em Matemática, Estatística, Programação e Negócios são mais relevantes para uma carreira em Ciência de Dados, pode-se atribuir diferentes pesos a fim de dar maior importância para estas avaliações. Imagine então que a média final da avaliação do candidato será uma média ponderada das suas notas com os pesos 2 para Matemática/Estatística e Programação, peso 3 para Negócios e 1 para Português. A sua média final será a soma das notas nas provas multiplicadas pelos seus respectivos pesos e dividida pela soma dos pesos, ficando 7,125 ($\frac{2\times8 + 2\times8 + 1\times4 + 3\times7}{2+2+1+3} = 7,125$). Com a média ponderada o candidato passaria para a próxima fase!

    Infelizmente, o Pandas não possui uma função ou método que calcule diretamente uma média ponderada. Contudo, outros pacotes com o Numpy possuem funções que permitem isso. De qualquer forma, realizar este cálculo sem uma função pronta também é simples.

    Vamos calcular a Média Final Ponderada das notas dos candidatos nas avaliações de duas formas: uma utilizando a função np.average() do Numpy e outra simplesmente realziando o cálculo da média ponderada com o Pandas.

In [126]:
# Importando o pacote Numpy
import numpy as np

# Criando um DataFrame apenas com as notas do candidatos e armazenando na variável notas
notas = df[['Nota Mat/Est', 'Nota Program', 'Nota Port', 'Nota Neg']]
# Criando uma variável com os respectivos pesos de cada avaliação
pesos = [2, 2, 1, 3]

# Calculando a média ponderada multiplicando sem uma função
df['Media Final Ponderada (Pandas)'] = (notas * pesos).sum(axis=1)/sum(pesos)
# Calculando a média ponderada com a função np.average() do Numpy
df['Media Final Ponderada (Numpy)'] = np.average(notas, axis=1, weights=pesos)
